Deploy บน GitHub Pages (ฟรี)
1
สมัคร GitHub
ไปที่ github.com แล้วกด Sign up — ฟรีทั้งหมด
→ github.com
2
สร้าง Repository ใหม่
กด New repository ตั้งชื่อว่า xau-lot-calc แล้วกด Create
3
อัปโหลดไฟล์ HTML
คลิก Add file → Upload files ลากไฟล์ gold-lot-calculator.html วาง แล้วเปลี่ยนชื่อเป็น index.html
4
เปิด GitHub Pages
ไปที่ Settings → Pages → Source เลือก main branch แล้วกด Save
5
ได้ Link แล้ว!
รอประมาณ 1–2 นาที จะได้ URL:
https://username.github.io/xau-lot-calc/
⚡ ทางเลือกเร็วกว่า — Netlify Drop
ไม่ต้องสมัคร แค่ลากไฟล์วางที่ netlify.com/drop ได้ link ภายใน 10 วินาที
→ app.netlify.com/drop
